# MoZhiMusicPlayer
仿制当前主流的某T的音乐软件，基于dotNET_C#_WPF的个人开源音乐客户端
# 功能：
  1.支持歌词字同步动画，同步歌词进度
  2.支持多歌手写真动画交叉轮播
  3.基本的音乐播放器功能（无需解释这些）
  4.基本的歌曲导入（选择导入，本地全盘扫描导入）
